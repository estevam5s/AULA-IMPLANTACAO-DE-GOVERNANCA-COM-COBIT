# Governança de TI com ITIL - Sistema Interativo de Aprendizagem

Sistema interativo completo para o curso "Governança de TI com ITIL" desenvolvido em React.

## 🎯 Sobre o Projeto

Este sistema foi desenvolvido para apoiar o ensino da disciplina de Governança de TI utilizando o framework ITIL, ministrada pela Dra. Eng. Maiara Heil na Estácio.

## 🚀 Como executar

```bash
# Instalar dependências
npm install

# Executar em modo de desenvolvimento
npm start

# Construir para produção
npm run build
